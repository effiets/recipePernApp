# Recipe App 

## A full stack Responsive Application for Recipes. 

### Technologies:

* Typescript
* ReactJS
* Bootrstap
* PostgreSQl
* ExpressJS

### Things the App can do:

* Bring All Database recipes
* Search by name or by course(Breakfast, Main Course etc)
* Add a new recipe
* Add new ingredients if database doesn't have
* Delete a recipe
